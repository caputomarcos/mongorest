MongoRest Installation
======================

MongoRest is on `PyPI <https://pypi.python.org/pypi/MongoRest>`_ so all you need
to do is:

.. code-block:: console

    $ pip install mongorest

Requirements
------------

MongoRest depends on the following requirements:

    * Cerberus >= 0.9.0
    * PyMongo >= 3.0.0
    * Six >= 1.10.0
    * Werkzeug >= 0.10.0
